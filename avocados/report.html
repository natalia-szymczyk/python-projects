Szereg rozdzielczy¶

Konstrukcja szeregu rozdzielczego dla zmiennej opisujcej redni cen awokado.

Liczb przedziaw obliczyam ze wzoru: $k \approx 1 + 3.22 log(n) $

Nastpnie do obliczenia szerokoci przedziau zastosowaam wzr: $h \approx \displaystyle\frac{x_{max} - x_{min}}{k} $

In [12]:
average_price <- data$average_price

n = length(average_price)
R = max(average_price) - min(average_price)
k = round( 1 + 3.22 * log(n))
h = round(R/k, 3)

Test statystyczny¶

Czy rednia cena awokado jest nisza ni 1.5?

Ze wzgldu na du prb oraz znan wariancj uyj testu Z. Zakadam, e rozkad jest normalny.
Bdzie to lewostronny test na poziomie istotnoci $\alpha = 0.05$.

$H_0$: $\mu = 1.5$
$H_1$: $\mu < 1.5$

Wzr na statystyk: $Z = \displaystyle\frac{\bar{X}-\mu_{0}}{\frac{\sigma}{\sqrt{n}}}$

In [16]:
alpha <- 0.05
mi <- 1.5

z_average_price = (mean(data_sample$average_price) - mi) / sqrt(var(data$average_price) / length(data_sample$average_price))

z_kr <- qnorm(alpha)

cat('Przedzia krytyczny: ( -inf ;', round(z_kr, 3), ') \n')
cat('Warto statystyki:     ', round(z_average_price, 3) ,'\n\n')

if(z_average_price < z_kr){
    cat('Warto statystyki zawiera si w obszarze krytycznym, dlatego naley odrzuci hipotez zerow na rzecz hipotezy alternatywnej. \nrednia cena awokado jest nisza ni', mi, '\n')
}else{
    cat('Warto statystyki nie zawiera si w obszarze krytycznym, dlatego nie ma podstaw do odrzucenia hipotezy zerowej. \nrednia cena awokado nie jest nisza ni', mi, '\n')
}
